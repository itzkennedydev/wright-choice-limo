# Wright Choice Limousine

Professional limousine and chauffeur services in Chicago and surrounding suburbs.

## About

Wright Choice Limousine provides premier transportation services including:

- Corporate travel
- Airport transfers (O'Hare & Midway)
- Special events
- Chauffeur services

Serving Chicago, Bloomingdale, Naperville, St. Charles, and Northwest Indiana.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
